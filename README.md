## configuration

#### global prefix

```
"easy.home": "/Users/xxxx/Desktop"
```

#### default prefix

when you just input a repo name
```
/Users/xxxx/easy
```

#### custom

just input full path with repo name
```
/Users/xxxx/aaa/bbb/ccc
```
